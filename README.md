# Zorin Dark Blue qBittorrent Theme

A clean, minimal qBittorrent theme inspired by **Zorin OS Dark Blue**.

This theme focuses on visual consistency, soft contrast, and a monochrome icon style that blends naturally with modern GNOME setups.

---

## ✦ Design Philosophy

* Deep neutral background (`#1e2529`)
* Soft blue foreground (`#bde6fb`)
* Monochrome SVG icons
* Subtle hover & selection states
* No visual noise, no unnecessary gradients

The goal was simple: make qBittorrent feel native on a Zorin-based desktop without breaking usability.

---

## ✦ Installation

1. Download the `.qbtheme` file from the releases section.
2. Open **qBittorrent**
3. Go to:
   `Tools → Options → Behavior`
4. Enable **Use custom UI theme**
5. Select the downloaded `.qbtheme` file
6. Restart qBittorrent

---

## ✦ Compatibility

* Tested on Linux (Wayland)
* Built for Qt-based qBittorrent versions supporting `.qbtheme`
* Optimized for dark desktop environments
